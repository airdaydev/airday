# Airday server exploration

This is v0 of Airday's backend in Rust.

## Likely dependencies
- tokio (async runtime)
- axum (web framework)
- serde (serialisation, deserialisation)
- chrono (tz package)
- chrono-tz (tz extension?)
- validator (validate requests)
- uuid (id generation)
- sqlx (sqlite)
- automerge (maybe)
